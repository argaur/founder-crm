# High-Yield Note Patterns

## Note Types That Reliably Produce LinkedIn Ideas

### 1. Case Study Research Docs (highest yield)
Files like `Blinkit Dark Store Case Study.md` and `CRM Case Study.md`.
- Contain specific data points (225 orders/hour, 7/10 founders, 94% intent vs 50% usage)
- Gaurav's own research — maximum authenticity
- Already include a "Relationship to LinkedIn Content" section in some files
- One case study typically yields 3–4 distinct post angles (ops lens, PM lens, data lens, framework lens)

### 2. Build Logs (high yield)
Files like `GWS CLI - Build Log.md`.
- Real decisions with real tradeoffs documented (why Python over n8n)
- Execution lessons from direct experience
- Strong authenticity score
- Best angle: build-vs-buy decisions, what the build taught about product thinking

### 3. Career/Strategy Planning Docs (high yield)
Files like `Career Plan - Must Do/Action Plan.md`.
- Meta-content angles ("you are the case study")
- Personal operating system as content
- Already has post ideas embedded ("write about what you're actually doing")
- Gaurav's own frameworks and habits

### 4. Newsletter Insights (moderate yield — filtered)
Files like `Knowledge/Newsletter Insights/2026-03-15.md`.
- Pre-extracted by Claude with relevance scores
- Strong on specificity (data points included)
- Lower authenticity — external source, not Gaurav's direct experience
- Best when Gaurav has a clear POV to add (e.g., xAI org failure → PM lesson)
- Skip insights that are just reporting without Gaurav's angle

### 5. About Me / Identity Notes (lower frequency, high specificity)
File: `About Me/About Me.md`.
- Career origin story (Architecture → PM) is a strong authenticity post
- Professional summary has specific metrics (30+ products, 40% efficiency gains)
- Use sparingly — one origin story post per quarter, not monthly

## Notes That Look Rich But Aren't (for LinkedIn)

- **Daily Call notes** — process notes, assignments, research method reminders. No publishable insight unless a specific framework is articulated.
- **Weekend Session list** — just titles, no content migrated
- **Books.md** — reading list only, no learnings yet
- **Notion Import folder** — mirrors of Notion pages, nothing original

## Signals That a Note Has LinkedIn Potential

- Contains a specific number or data point
- Contains a week-by-week or stage-by-stage breakdown
- Contains a named tradeoff ("X vs Y")
- Has a "Relationship to LinkedIn Content" section
- Gaurav explicitly flagged it in Action Plan as a post to write
- The note describes something Gaurav built or researched directly
